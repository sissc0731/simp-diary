<h1 align="center">🐶 舔狗日记</h1>
<h3 align="center">AI 舔狗文学生成器 — 记录每一份自我感动的深情</h3>

<p align="center">
  <i>"她今天回我了。虽然只回了一个'嗯'，但只回了我一个人。" 🥲</i>
</p>

---

## 📖 这是什么

上传你和 TA 的聊天记录（或者描述你的故事），Claude Code 自动生成**舔狗文学风格日记**。

每篇日记都包含：
- 📅 日期和天气（心情决定天气）
- 💬 今日互动记录
- 💸 今日付出统计
- 💭 内心独白
- 🌟 自我感动指数

---

## 🎭 示例

### 输入
```
她是我同事，长头发，笑起来很好看。
今天她让我帮她带了一杯奶茶，还说了声谢谢。
昨天我帮她做了PPT，加班到凌晨两点。
她好像有个异地男友，但我觉得我还有机会。
我每天都会给她的朋友圈点赞，虽然她从来没回过我。
```

### 输出

```
━━━━━━━━━━━━━━━━━━
📅 2026年6月16日 | ☀️ 晴（她对我笑了，世界就是晴天）
━━━━━━━━━━━━━━━━━━

💬 今日互动：

她说："帮我带杯奶茶"
我说："好！什么口味？"
她说："随便"
我买了三种口味让她挑，她选了最便宜的那种。
她真好，会替我省钱。

💸 今日付出：
• 奶茶 ×3：¥45
• 帮她改PPT：0元（她没要求，我主动的）
• 朋友圈点赞：无价
• 合计：¥45 + 一颗真心

💭 深夜emo：

凌晨两点，PPT终于改完了。每一页动画我都反复调试，虽然她可能根本不会注意到。发给她的那一刻，感觉自己像完成了一件艺术品。

她回了一个"谢谢🌹"。

那个玫瑰花emoji，我看了十七遍。

她有男朋友又怎样？PPT是我帮她做的。这份心意，是独一无二的。我相信时间会证明一切。总有一天她会知道，真正懂她的人，是我。

🌟 自我感动指数：★★★★★（满分）
📊 她回我字数统计：3个字
📊 我回她字数统计：约500字
📊 投入产出比：无法计算，感情不能用数字衡量

━━━━━━━━━━━━━━━━━━
💔 今日舔狗语录：
"她只是把我当同事，我却把她当余生。"
━━━━━━━━━━━━━━━━━━
```

## 🚀 快速开始

```bash

# 下载 Skill

cp -r simp-diary/.claude ./
```
